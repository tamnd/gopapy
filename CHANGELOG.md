# Changelog

All notable changes to gopapy are recorded here. Per-version notes
live in [`changelog/`](changelog/) — each release has its own file
(`changelog/x.y.z.md`) so the history grows by adding files rather
than editing one ever-growing document.

The format follows
[Keep a Changelog 1.1](https://keepachangelog.com/en/1.1.0/). Once
gopapy reaches 1.0 the project will follow
[Semantic Versioning](https://semver.org/spec/v2.0.0.html); until
then, expect minor version bumps to sometimes include breaking
changes.
