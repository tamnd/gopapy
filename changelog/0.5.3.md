added, covering async I/O (aiohttp, anyio), HTTP (httpx, httpcore, urllib3),

## Fixes

Two issues discovered during corpus expansion:

### match soft keyword with subscript target

`match["key"] = value` was incorrectly parsed as a match statement.
`looksLikeMatchStmt` returned true when the next token after `match` was
`[`, unconditionally. Fixed by requiring `:` after the closing `]`: the
`tkLBrack` case now calls `parenFollowedByColon()` (which already handles
`]` at depth 0) instead of returning true immediately. Fixes
`paramiko/config.py`.

### `ur"..."` in example file

`pygments/tests/examplefiles/python/unicodedoc.py` uses the `ur"..."` string
prefix (Python 2 syntax, invalid in Python 3). Added `unicodedoc.py` to
`isIntentionalBadFixture` so `gopapy check` skips it (both gopapy and CPython
reject it).
