## [0.2.14] - 2026-04-28

### Added

- **`Module.TypeIgnores` field.** CPython's `ast.Module` carries a
  `type_ignores` field listing every `# type: ignore` comment that
  appears in the file. The gopapy `Module` struct now has the matching
  `TypeIgnores []TypeIgnore` field so the struct layout matches what
  tools that walk the AST expect. The list is always empty until Phase
  3 adds comment collection, but the field must exist for structural
  parity with CPython.
- `TypeIgnore` struct with `Lineno int` and `Tag string` fields,
  mirroring `ast.TypeIgnore`.

### Changed

- CLI version bumped to `0.2.14`.
