## [0.3.9] - 2026-04-28

Phase 1.5, Round 9 of the parse-every-Python roadmap.

### Added

- **`TestOracle_Py38` — Python 3.8 oracle test (active, no skip).**
  `gopapy dump --py 3.8` is now verified byte-identical to CPython 3.8's
  `ast.dump()` for all compatible fixtures: 69 pass, 37 skipped (3.9+/3.10+/
  3.11+/3.12+/3.13+/3.14+ syntax), 0 fail.

- **ExtSlice support for Python 3.8 multi-dimensional subscripts.**
  In Python 3.8, `a[1:2, 3]` and `m[:, :, 0:1]` use `ExtSlice(dims=[...])`
  when the subscript tuple contains at least one `Slice`; subscript tuples
  with no `Slice` elements are wrapped in `Index(value=Tuple(...))` instead.

- **ExceptHandler `type=None, name=None` for bare `except:` in Python 3.8.**
  A bare `except:` clause now emits `ExceptHandler(type=None, name=None, body=[...])`
  under `--py 3.8`, matching CPython 3.8's ast.dump() output.

### Changed

- CLI version bumped to `0.3.9`.
