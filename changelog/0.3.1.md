## [0.3.1] - 2026-04-28

Phase 1.5, Round 1 of the parse-every-Python roadmap
(`~/notes/Spec/1200/1238_gopapy_every_python_roadmap.md`).

### Added

- **`stdlib-compat` CI job.** New matrix job that runs `gopapy check`
  and `gopapy unparse --check` against the standard library of each
  Python version from 3.9 through 3.13. Currently informational
  (`continue-on-error: true`) so it does not block merges; each row
  will be flipped to a hard gate once that version reaches zero
  failures (Round 1.5.5 / v0.3.5). The matrix uses `fail-fast: false`
  so all five rows always run to completion and the full picture is
  visible in a single CI run.

### Changed

- CLI version bumped to `0.3.1`.
